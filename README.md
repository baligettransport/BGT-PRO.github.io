# BGT-PRO.github.io
Tentang aplikasi ini:

Jika Anda adalah anggota atau pembuat komunitas yang dihosting di GroupApp, aplikasi seluler kami memungkinkan Anda mengikuti diskusi komunitas yang sedang berlangsung, mengawasi acara mendatang, dan menerima pemberitahuan dari komunitas Anda saat bepergian.

Keamanan Data:

Keamanan dimulai dengan memahami cara developer mengumpulkan dan membagikan data Anda. Praktik privasi dan keamanan data dapat bervariasi berdasarkan penggunaan, wilayah, dan usia Anda. Developer memberikan informasi ini dan dapat memperbaruinya seiring waktu.
